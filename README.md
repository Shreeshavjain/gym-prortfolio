# IRON PULSE FITNESS - Gym Portfolio Template

A professional, modern gym portfolio website built with Next.js, TypeScript, Tailwind CSS, and Framer Motion. This template includes all essential sections for a fitness business website with smooth animations and mobile responsiveness.

## ‚ú® Features

- **Hero Section** - Eye-catching landing with responsive navbar and smooth navigation
- **Programs Section** - Horizontally scrolling program cards with descriptions
- **Achievements Section** - Statistics showcase with responsive grid
- **Trainers Section** - Team member showcase with horizontal scroll
- **Membership Plans** - Three-tier pricing with WhatsApp integration
- **Contact Section** - Google Maps integration and social media links
- **Mobile Menu** - Hamburger navigation for mobile devices
- **Smooth Animations** - Framer Motion section animations with prefers-reduced-motion support
- **Premium Styling** - Dark theme with professional branding
- **Social Media Integration** - Simple Icons CDN for official brand icons
- **WhatsApp Integration** - Direct messaging links on CTAs

## üöÄ Getting Started

### Prerequisites

- Node.js 16+ 
- npm or yarn

### Installation

```bash
# Clone the repository
git clone <repository-url>

# Navigate to project directory
cd gym-template

# Install dependencies
npm install
# or
yarn install
```

### Development

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) to view the website.

### Production Build

```bash
npm run build
npm start
```

## üìù Customization

### Colors & Branding

Edit `app/globals.css` and `app/page.tsx` to customize:
- Primary colors
- Accent colors
- Typography
- Spacing

### Content

Replace placeholder content in `app/page.tsx`:
- Gym name and tagline
- Programs and descriptions
- Trainer information
- Pricing details
- Contact information

### Images

Place local images in `public/images/` directory and reference them in the code.
